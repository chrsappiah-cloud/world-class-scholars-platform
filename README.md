# World Class Scholars Platform

Production-ready Node.js platform for World Class Scholars, serving dementia care resources, courses, and campaigns. Built with native HTTP, clustering, and Supabase/Postgres integration. Responsive HTML/CSS/JS frontend. Ready for deployment to Vercel or Railway.

## Features
- Native Node.js HTTP server
- Static and dynamic routes
- Supabase/Postgres integration (placeholder)
- Clustering for scaling
- Responsive frontend (HTML/CSS/JS)
- Production scripts and linting

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```
2. Start development server:
   ```sh
   npm run dev
   ```
3. Start production server:
   ```sh
   npm start
   ```

## Deployment
- Vercel: `vercel --prod`
- Railway: `railway deploy`
- AWS EC2: Use PM2 and Nginx
